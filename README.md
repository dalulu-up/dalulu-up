## 🛠️ 精选项目

### 💬 [reply-tool](https://github.com/dalulu-up/reply-tool) — AI 智能回复工具

> 双击快捷键 → 本地大模型秒回粉丝消息，不用打开任何 App

- **核心功能**：Hammerspoon 监听快捷键，调用本地 Qwen3 模型，自动生成回复
- **双模式**：自由回复（右Option×2）/ 三步粉丝流程（右Command×2）
- **技术栈**：Python · MLX · Hammerspoon · OCR
- **亮点**：完全本地运行，零 API 费用，支持自定义话术模板

---

### 🌡️ [valuation-monitor](https://github.com/dalulu-up/valuation-monitor) — 估值温度计

> 输入股票代码，一秒看 PE/PB 在历史分位——现在贵还是便宜

- **核心功能**：查询 A 股 / 港股 / 美股的估值历史分位，辅助买卖决策
- **技术栈**：Python · Flask · AkShare · Pandas
- **亮点**：Web 界面 + 手机同局域网访问，数据自动缓存，双击即启动

---

### 📡 [news-radar](https://github.com/dalulu-up/news-radar) — 投资新闻采集器

> 每天自动抓 AI / 芯片 / 数据中心新闻，本地大模型精选 + 思考题，推送手机

- **核心功能**：DuckDuckGo 精准搜索 + RSS 聚合 → 本地 Qwen3 分析 → Bark 推手机 + Obsidian 同步
- **数据源**：华尔街见闻、财联社、SemiAnalysis、量子位、HackerNews、AINews...
- **技术栈**：Python · DuckDuckGo Search · BeautifulSoup · LaunchAgent
- **亮点**：可信度标注（官方/报道/传闻）；熵减设计，每天只看 3-5 条精华

---

### 📋 [follow-builders](https://github.com/zarazhangrui/follow-builders) — AI Builders 日报

> 追踪 25+ 顶级 AI builder 的 X / 播客动态，每日自动生成摘要

- **核心功能**：聚合顶级 builder 的 Twitter 和播客内容，Claude 提炼成可读摘要
- **技术栈**：Node.js · Claude API · cron
- **亮点**：跟 builder，不跟 influencer；零 API key 开箱即用
